# Support

## Supporting Loadable components

Loadable components is a MIT-licensed open source project. It's an independent project with ongoing development made possible thanks to the support of these awesome [backers](/BACKERS.md). If you'd like to join them, please consider:

- [Become a backer or sponsor on OpenCollective](https://opencollective.com/loadable).

## Gold Sponsors

Gold Sponsors are those who have pledged \$100/month and more to Loadable Components.
